# 🌟Project Name: Friendship Analytics

## 📌Project Description:

Friendship Analytics is a web application that helps users track and analyze their interactions with friends. It provides a visual breakdown of communication types such as text, calls, and video interactions, making it easier to understand relationship patterns.

## 🚀Technologies Used:
### 1.Next.js
### 2.React
### 3.Tailwind CSS
### 4.Recharts
### 5.JavaScript (ES6)

## ✨Key Features:
### 1.Visual analytics using interactive pie charts
### 2.Tracks different interaction types (Text, Call, Video)
### 2.Stores data user friendly
